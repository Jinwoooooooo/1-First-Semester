Inline Tag 연습

    

    
진하게 표시하려면 <b>진하게 표시</b> 태그를 이용한다.

    
중요한 내용을 표시하려면 <strong>중요한 내용</strong> 태그를 이용한다.

    
강조를 하려면 <em>강조</em> 태그를 이용한다.

    
이탤릭체로 표현하려면 <i>이탤릭체로 표현</i> 태그를 이용한다.

    
작은 문자로 표현하려면 <small>작은 문자로</small> 태그를 이용한다.

    
내용을 삭제하려면 <del>삭제한 내용으로 표시</del> 태그를 이용한다.

    
윗첨자를 하려면 <sup>윗첨자</sup> 태그를 이용한다.

    
아래첨자를 하려면 <sub>아래첨자</sub> 태그를 이용한다.

    
하이라이팅을 하려면 <mark>하이라이팅</mark> 태그를 이용한다.

    
이차방정식 : ax2 + bx + c = 0
